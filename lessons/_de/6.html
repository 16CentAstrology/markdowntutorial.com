In diesem Tutorial geht es um die Erstellung von Listen in Markdown.

Es gibt zwei Arten von Listen im bekannten Universum: ungeordnete und geordnete. Das ist eine schicke Art zu sagen, dass es Listen mit Aufzählungspunkten und Listen mit Zahlen gibt.

Um eine ungeordnete Liste zu erstellen, müssen Sie jedem Eintrag in der Liste ein Sternchen ( * ) voranstellen und dann ein Leerzeichen folgen lassen. Jedes Element der Liste erhält außerdem eine eigene Zeile. Eine Lebensmittelliste in Markdown könnte zum Beispiel so aussehen: 

* Milch
* Eggs
* Lachs
* Butter

Diese Markdown-Liste würde in die folgenden Aufzählungspunkte umgewandelt:

  • Milch
  • Eggs
  • Lachs
  • Butter

Machen Sie aus den durch ein Komma getrennten Wörtern in dem unten stehenden Kasten eine Liste.

So geht's! So schreiben Sie eine ungeordnete Liste. Jetzt wollen wir über geordnete Listen sprechen.

Einer geordneten Liste werden Zahlen anstelle von Sternchen vorangestellt. Schauen Sie sich dieses Rezept an:

  1. Drei Eier in eine Schüssel aufschlagen
  2. Gießen Sie einen halben Liter Milch in die Schüssel
  3. Den Lachs gründlich mit Butter einreiben
  4. Den Lachs in die Eier-Milch-Schüssel geben

Um das in Markdown zu schreiben, müssten Sie Folgendes tun:

1. Drei Eier in eine Schüssel aufschlagen
2. Gießen Sie einen halben Liter Milch in die Schüssel
3. Den Lachs gründlich mit Butter einreiben
4. Den Lachs in die Eier-Milch-Schüssel geben

Einfach, nicht wahr? Es sieht genau so aus, wie Sie es von einer Liste erwarten würden.

Verwandeln Sie den Rest des Rezepts in eine geordnete Liste in dem Feld unten.

Gute Arbeit!

Sie können kursiv, fett oder Links innerhalb von Listen hinzufügen, wie Sie es erwarten würden. Machen Sie im Feld unten die lateinischen Namen der Pflanzen kursiv.

Herrlich!

Gelegentlich werden Sie das Bedürfnis haben, eine Liste mit mehr Tiefe zu erstellen oder eine Liste in einer anderen zu verschachteln. Keine Angst, die Markdown-Syntax ist genau dieselbe. Sie müssen nur daran denken, jedes Sternchen um ein Leerzeichen mehr einzurücken als den vorhergehenden Eintrag.

In der folgenden Liste fügen wir beispielsweise jedem „Haupt“-Listenelement einige Unterlisten hinzu, in denen die Personen detailliert beschrieben werden: 

* Tim
 * Ein Reporter
 * Hat orangefarbenes Haar
 * Freunde mit dem tollsten Hund der Welt
* Haddock
 * Ein Schiffskapitän
 * Hat einen fantastischen Bart
 * Liebt Whiskey
   * Möglicherweise auch Scotch?

Bei der Darstellung wird diese Liste zu folgender Gruppierung:

  • Tim
    • Ein Reporter
    • Hat orangefarbenes Haar
    • Freund mit dem tollsten Hund der Welt
  • Haddock
    • Ein Schiffskapitän
    • Hat einen fantastischen Bart
    • Liebt whiskey
      • Möglicherweise auch Scotch?

Verwandeln Sie die Merkmale des Charakters in Unterpunkte in dem Feld unten.

Unglaublich! Sie können die Einrückung und das Hinzufügen von Unterlisten zwar unbegrenzt fortsetzen, aber in der Regel ist es eine gute Idee, nach drei Ebenen aufzuhören, da Ihr Text sonst unübersichtlich wird.

Es gibt noch einen weiteren Trick für Listen und Einrückungen, den wir erforschen werden, und der den Fall von Absätzen betrifft. Nehmen wir an, Sie möchten eine Aufzählungsliste erstellen, die einen zusätzlichen Kontext erfordert (aber keine weitere Liste). Sie könnte zum Beispiel so aussehen:

  1. Drei Eier in eine Schüssel aufschlagen.

    Nun müssen Sie die Eier so aufschlagen, dass keine Schweinerei entsteht.

    Wenn Sie eine Schweinerei machen, benutzen Sie ein Handtuch, um sie aufzuräumen!

  2. Gießen Sie einen halben Liter Milch in die Schüssel.

    Grundsätzlich gilt derselbe Leitfaden wie oben: Sei nicht unordentlich, aber wenn du es bist, räume es auf!

  3. Den Lachs kräftig mit Butter einreiben.

    Mit „kräftig“ meinen wir eine streng vertikale Bewegung. Wie Julian einmal sagte:

    Hoch und runter und rundherum, so geht Butter auf Lachs.

  4. Den Lachs in die Eier-Milch-Schüssel geben.

    Hier einige Techniken zum Absetzen von Lachsen:

    • Stellen Sie sicher, dass keine Kinder anwesend sind.
    • Beide Hände benutzen.
    • Halten Sie immer ein Handtuch bereit, falls Sie etwas verschmutzt haben.

Um diese Art von Text zu erstellen, muss Ihr Absatz in einer eigenen Zeile unterhalb des Aufzählungspunkts beginnen und um mindestens ein Leerzeichen eingerückt sein. Die obige Liste sieht in Markdown zum Beispiel so aus: 

1. Drei Eier in eine Schüssel aufschlagen.

 Nun müssen Sie die Eier so aufschlagen, dass keine Schweinerei entsteht.

 Wenn du eine Schweinerei machst, benutze ein Handtuch, um sie aufzuräumen!

2. Gießen Sie einen halben Liter Milch in die Schüssel.

 Grundsätzlich gilt derselbe Leitfaden wie oben: Seien Sie nicht unordentlich, aber wenn Sie es sind, räumen sie auf!

3. Den Lachs kräftig mit Butter einreiben.

   Mit „kräftig“ meinen wir eine streng vertikale Bewegung. Wie Julian einmal sagte:
   > Hoch und runter und rundherum, so geht Butter auf Lachs.

4. Den Lachs in die Eier-Milch-Schüssel geben.

   Hier einige Techniken zum Absetzen von Lachsen:

   * Stellen Sie sicher, dass keine Kinder anwesend sind.
   * Beide Hände benutzen.
   * Halten Sie immer ein Handtuch bereit, falls Sie etwas verschmutz haben.

Beachten Sie, dass die ersten beiden Einträge ein einzelnes Leerzeichen haben. Das sieht etwas seltsam aus, daher sollten Sie die Zeichen entsprechend einrücken (wie bei den Punkten drei und vier). In diese Absätze können Sie alle möglichen anderen Markdown-Elemente einfügen, z. B. Blockzitate oder sogar andere Listen!

Wandeln Sie die Aufzählungspunkte im folgenden Kasten in eigene Absätze um.
