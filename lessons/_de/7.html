Markdown bietet mehrere Möglichkeiten zur Formatierung von Absätzen.

Nehmen wir als Beispiel ein paar Gedichtzeilen. Angenommen, Sie wollen einen Text schreiben, der wie folgt aussieht:

Widerspreche ich mir selbst?
Sehr wohl, dann widerspreche ich mir selbst,
(Ich bin groß, ich enthalte viele.)

Nun könnte man meinen, dass die Eingabe jedes Verses in eine eigene Zeile ausreichen würde, um das Problem zu lösen: 

Widerspreche ich mir selbst?
Sehr wohl, dann widerspreche ich mir selbst,
(Ich bin groß, ich enthalte viele.)

Leider wäre das falsch! Dieses Markdown würde einfach als eine einzige gerade Linie dargestellt: Widerspreche ich mir selbst? Sehr wohl, dann widerspreche ich mir selbst, (Ich bin groß, ich enthalte viele.).

Wenn Sie zwangsweise eine neue Zeile einfügen, brechen Sie die Zusammengehörigkeit auf: 

Widerspreche ich mir selbst?

Sehr wohl, dann widerspreche ich mir selbst,

(Ich bin groß, ich enthalte viele.)

Dies ist ein so genannter hard break; in unserer Dichtung ist ein soft break erwünscht. Sie können dies erreichen, indem Sie nach jeder neuen Zeile zwei Leerzeichen einfügen. Das kann man nicht sehen, da Leerzeichen unsichtbar sind, aber es würde etwa so aussehen: 

Widerspreche ich mir selbst?··
Sehr wohl, dann widerspreche ich mir selbst,··
(Ich bin groß, ich enthalte viele.)

Jeder Punkt ( · ) steht für ein Leerzeichen auf der Tastatur.
