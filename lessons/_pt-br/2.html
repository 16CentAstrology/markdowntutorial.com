Vamos dar uma olhada em outra convenção de formatação: o cabeçalho. Cabeçalhos são frequentemente usados em sites, artigos de revistas e avisos, para chamar atenção para uma seção. Como o nome sugere, eles atuam como títulos ou subtítulos acima das seções.

Existem seis tipos de cabeçalhos, em tamanhos decrescentes:

Este é o cabeçalho um

Este é o cabeçalho dois

Este é o cabeçalho três

Este é o cabeçalho quatro

Este é o cabeçalho cinco
Este é o cabeçalho seis

Para fazer cabeçalhos no Markdown, você prefixa a frase com um símbolo de cerquilha (#). Você coloca o mesmo número de cerquilhas conforme o tamanho do cabeçalho que deseja. Por exemplo, para um cabeçalho de tamanho um, você usaria uma cerquilha (# Cabeçalho Um), enquanto para um cabeçalho três, você usaria três (### Cabeçalho Três).

Para esta próxima lição, faça cada cabeçalho do tamanho correto.

Ótimo!

Cabe a você decidir quando é apropriado usar qual cabeçalho. Em geral, cabeçalhos um e seis devem ser usados com moderação.
