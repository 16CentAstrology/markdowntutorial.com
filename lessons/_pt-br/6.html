Este tutorial é sobre como criar listas em Markdown.

Existem dois tipos de listas no universo conhecido: não ordenadas e ordenadas. Isso é uma maneira chique de dizer que existem listas com pontos e listas com números.

Para criar uma lista não ordenada, você deve prefixar cada item da lista com um asterisco ( * ), seguido por um espaço. Cada item da lista também recebe sua própria linha. Por exemplo, uma lista de compras em Markdown pode parecer assim: 

* Leite
* Ovos
* Salmão
* Manteiga

Esta lista em Markdown seria renderizada nos seguintes pontos de bala:

  • Leite
  • Ovos
  • Salmão
  • Manteiga

Tudo certo! É assim que você escreve uma lista não ordenada. Agora, vamos falar sobre listas ordenadas.

Uma lista ordenada é prefixada com números, em vez de asteriscos. Veja esta receita:

  1. Quebre três ovos sobre uma tigela
  2. Despeje um litro de leite na tigela
  3. Esfregue o salmão vigorosamente com manteiga
  4. Coloque o salmão na tigela de leite com ovos

Para escrever isso em Markdown, você faria assim:

1. Quebre três ovos sobre uma tigela
2. Despeje um litro de leite na tigela
3. Esfregue o salmão vigorosamente com manteiga
4. Coloque o salmão na tigela de leite com ovos

Fácil, certo? É exatamente como você esperaria que uma lista parecesse.

Ocasionalmente, você pode achar necessário fazer uma lista com mais profundidade, ou, para aninhar uma lista dentro de outra. Não tenha medo, porque a sintaxe do Markdown é exatamente a mesma. Tudo o que você precisa fazer é lembrar de indentar cada asterisco um espaço a mais do que o item anterior.

Por exemplo, na lista a seguir, vamos adicionar algumas sub-listas a cada item "principal" da lista, descrevendo as pessoas em detalhes: 

* Tintim
 * Um repórter
 * Tem cabelo laranja fofo
 * Amigo do cachorro mais incrível do mundo
* Haddock
 * Um capitão do mar
 * Tem uma barba fantástica
 * Ama uísque
   * Possivelmente também escocês?

Quando renderizada, esta lista se transforma no seguinte agrupamento:

  • Tintim
    • Um repórter
    • Tem cabelo laranja fofo
    • Amigo do cachorro mais incrível do mundo
  • Haddock
    • Um capitão do mar
    • Tem uma barba fantástica
    • Ama uísque
      • Possivelmente também escocês?

Estupendo! Embora você possa continuar a indentar e adicionar sub-listas indefinidamente, geralmente é uma boa ideia parar após três níveis; caso contrário, seu texto se torna uma bagunça.

Há mais um truque para listas e indentação que vamos explorar, e isso lida com o caso de parágrafos. Suponha que você queira criar uma lista com marcadores que exija algum contexto adicional (mas não outra lista). Por exemplo, pode parecer assim:

  1. Quebre três ovos sobre uma tigela.

    Agora, você vai querer quebrar os ovos de uma maneira que não faça bagunça.

    Se você fizer bagunça, use uma toalha para limpar!

  2. Despeje um litro de leite na tigela.

    Basicamente, siga as mesmas orientações acima: não faça bagunça, mas se fizer, limpe!

  3. Esfregue o salmão vigorosamente com manteiga.

    Por "vigorosamente", queremos dizer um movimento estritamente vertical. Julia Child uma vez disse:

    Para cima e para baixo e ao redor, é assim que a manteiga vai no salmão.

  4. Coloque o salmão na tigela de leite com ovos.

    Aqui estão algumas técnicas sobre como colocar o salmão:

    • Certifique-se de que não há trutas ou crianças presentes
    • Use as duas mãos
    • Sempre tenha uma toalha por perto em caso de bagunça

Para criar esse tipo de texto, seu parágrafo deve começar em uma linha sozinho abaixo do ponto da lista e deve ser indentado por pelo menos um espaço. Por exemplo, a lista acima parece assim em Markdown: 

1. Quebre três ovos sobre uma tigela.

 Agora, você vai querer quebrar os ovos de uma maneira que não faça bagunça.

 Se você _fizer_ bagunça, use uma toalha para limpar!

2. Despeje um litro de leite na tigela.

 Basicamente, siga as mesmas orientações acima: não faça bagunça, mas se fizer, limpe!

3. Esfregue o salmão vigorosamente com manteiga.

   Por "vigorosamente", queremos dizer um movimento estritamente vertical. Julia Child uma vez disse:
   > Para cima e para baixo e ao redor, é assim que a manteiga vai no salmão.
4. Coloque o salmão na tigela de leite com ovos.

   Aqui estão algumas técnicas sobre como colocar o salmão:

   * Certifique-se de que não há trutas ou crianças presentes
   * Use as duas mãos
   * Sempre tenha uma toalha por perto em caso de bagunça

Observe que os dois primeiros itens têm um único espaço. Isso parece um pouco estranho, então você pode querer indentar corretamente para alinhar os caracteres (como itens três e quatro). Nesses parágrafos, você pode incluir todos os tipos de outros elementos Markdown, como citações, ou até mesmo outras listas!
