Se você sabe como criar links no Markdown, também pode criar imagens. A sintaxe é quase a mesma.

As imagens também têm dois estilos, assim como os links, e ambos são renderizados da mesma maneira. A diferença entre links e imagens é que as imagens são precedidas por um ponto de exclamação ( ! ).

O primeiro estilo de imagem é chamado de link de imagem inline. Para criar um link de imagem inline, insira um ponto de exclamação ( ! ), envolva o texto alternativo entre colchetes ( [ ] ) e depois envolva o link entre parênteses ( ( ) ). (O texto alternativo é uma frase ou sentença que descreve a imagem para deficientes visuais.)

Por exemplo, para criar um link de imagem inline para https://octodex.github.com/images/bannekat.png, com um texto alternativo que diz, Benjamin Bannekat, você escreveria isso no Markdown: ![Benjamin Bannekat](https://octodex.github.com/images/bannekat.png).

Na caixa abaixo, transforme o link em uma imagem e preencha os colchetes do texto alternativo para dizer "Um tigre bonito":

Maravilhoso!

Embora você não precise adicionar texto alternativo, isso tornará seu conteúdo acessível ao seu público, incluindo pessoas com deficiência visual, que usam leitores de tela ou que não têm conexões de internet de alta velocidade.

Para uma imagem de referência, você seguirá o mesmo padrão de um link de referência. Você precederá o Markdown com um ponto de exclamação, depois fornecerá dois colchetes para o texto alternativo e depois mais dois para a tag da imagem, assim: ![O pai fundador][Pai] Na parte inferior da sua página Markdown, você definirá uma imagem para a tag, assim: [Pai]: http://octodex.github.com/images/founding-father.jpg.

Na caixa abaixo, começamos a colocar algumas imagens de referência; você precisará completá-las, assim como na última lição. Chame a primeira tag de referência de "Preto" e faça um link para https://upload.wikimedia.org/wikipedia/commons/a/a3/81_INF_DIV_SSI.jpg; faça a segunda imagem linkar para http://icons.iconarchive.com/icons/google/noto-emoji-animals-nature/256/22221-cat-icon.png.

Ta da! Você aprendeu tudo sobre como adicionar imagens no Markdown!
