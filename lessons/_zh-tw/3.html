我們將學習如何連結到其他網站。

Markdown 裡有兩種連結，它們最終呈現的效果都相同。我們要介紹的第一種方式稱為 行內連結。建立行內連結的方式如下：將要顯示的文字輸入在方括號 ( [ ] )內，並將連結網址包在括號 ( ( ) )內。比方說，你想要建立一個前往 www.github.com 的超連結，並讓其顯示的文字為「前往 GitHub!」在 Markdown 中會寫為： [前往 GitHub!](www.github.com)。

在下方的文字框，建立一個顯示為「Search for it.」並前往 www.google.com 的連結。

做得好！

你也可以讓連結文字以斜體、粗體顯示。在下方的文字框，將「really, really」改為粗體，並把整句連結至 www.dailykitten.com。 記得將粗體的標記包在連結文字中。

非常棒！

雖然看起來有點怪，但其實你也可以在標題中放入連結。

將文字設為四號標題，並把「the BBC」連結至 www.bbc.com/news，以前往課程的下一步。

That's all there is to writing inline links.

另一種形式的連結稱為 參照連結。如同字面意思，這種連結實際上會參照文件中其他地方的內容。看不懂沒關係，考慮下方這個例子：

     Here's [a link to something else][another place].
     Here's [yet another link][another-link].
     And now back to [the first link][another place].

     [another place]: www.github.com
     [another-link]: www.google.com

例子中的「參照」是每行的第二組方括號：[another place] 及 [another-link]. 在 Markdown 文件的底部，這些方括號才被賦予確切網址的正式定義。這種參照連結的方式有個好處，就是多個指向相同地方的連結，可以使用標籤代稱，之後僅需統一對其賦予定義。比方說，我們決定將所有 [another place] 連結指向其他網站，不必一一修改，僅需改變參照本身的定義就好。

參照本身的定義內容，不會被彩現。它的格式為包著標籤的方括號，接著冒號，接著網址。

在下方的文字框，有一些寫到一半的參照連結，你的任務是將它們完成！將第一個參照標籤寫為「a fun place」，並讓它連結到 www.zombo.com；將第二個連結指向 www.stumbleupon.com。

那麼現在你已經了解如何在 Markdown 連結到其他網站了。
